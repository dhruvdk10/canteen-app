# Prompts and AI Tools Used

This document lists all the AI tools and prompts I used while developing the Canteen App project.

## Tools Used

- **ChatGPT (OpenAI)** – Assisted in coding, debugging, and writing explanations.
- **GitHub Copilot (optional)** – Suggested code snippets and structure during development.

## Example Prompts Used

1. **React & JSX Help**
   - "How to create a responsive cart page in React with quantity buttons and total price calculation?"
   - "How to add a veg/non-veg indicator dot inside a snack card in React?"

2. **API and Mock Data**
   - "How to fetch mock data from `db.json` using Axios in React?"
   - "How to simulate backend orders using `localStorage` in React?"

3. **Styling and Layout**
   - "How to align image, name, price, quantity, and remove button properly in a React cart component?"
   - "How to make a fixed-size image with object-fit cover in a responsive card?"

4. **Git & Version Control**
   - "How to ignore remote changes and keep only local changes in Git?"
   - "How to resolve merge conflicts in Git when MERGE_HEAD exists?"

5. **General Assistance**
   - Writing README.md and project documentation.
   - Explaining setup instructions and libraries used.
